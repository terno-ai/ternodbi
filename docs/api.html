API Reference

The TernoDBI API is RESTful and designed for easy consumption by AI agents.

Query Service

Method Endpoint Description
GET /api/query/health/ Service health check
GET /api/query/datasources/ List available datasources
POST /api/query/datasources/{id}/query/ Execute a SQL query

Admin Service

Method Endpoint Description
POST /api/admin/datasources/ Add a new datasource
GET /api/admin/datasources/{id}/sync/ Triger metadata sync
