MCP Integration

TernoDBI is built from the ground up to support the Model Context Protocol (MCP). This allows AI agents like Claude Desktop to interact with your data seamlessly.

Using the CLI

TernoDBI provides a dedicated command to output your MCP configuration snippet.

ternodbi mcp-config

Claude Desktop Configuration

Add the following to your claude_desktop_config.json:

{
  "mcpServers": {
    "ternodbi": {
      "command": "uvx",
      "args": ["--from", "terno-dbi", "dbi-mcp", "query"],
      "env": {
        "TERNODBI_API_URL": "http://localhost:8376",
        "TERNODBI_API_KEY": "YOUR_TOKEN"
      }
    }
  }
}

Tools Provided

  • list_tables: List tables available to the agent.
  • describe_table: Get columns and metadata for a table.
  • execute_query: Run a secure SQL query.
