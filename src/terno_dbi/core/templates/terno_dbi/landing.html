Key Features

🔐

Token Authentication

Secure API access with admin and query tokens. Create, revoke, and manage tokens with full audit trails.

🗄️

Multi-Database Support

Connect to PostgreSQL, MySQL, BigQuery, Snowflake, Databricks, Oracle, and SQLite with a unified API.

🛡️

Access Control

Fine-grained permissions with role-based access, table/column hiding, and row-level security filters.

📊

Query Execution

Execute SQL queries with pagination, export results as CSV, and get results as Parquet for analytics.

⚙️

Schema Management

Rename tables and columns, update descriptions, and manage metadata through the Admin API.

🔌

Easy Integration

Install as a Django app in any project or run as a standalone server. Full REST API documentation.

API Endpoints

Method Endpoint Description
GET /dbi/query/health/ Health check (no auth)
GET /dbi/query/datasources/ List all datasources
GET /dbi/query/datasources/{id}/tables/ List tables in a datasource
POST /dbi/query/datasources/{id}/query/ Execute SQL query
POST /dbi/admin/tokens/create/ Create new API token
GET /dbi/admin/tokens/ List all tokens
PATCH /dbi/admin/tables/{id}/ Update table metadata
DELETE /dbi/admin/tokens/{id}/ Delete a token
